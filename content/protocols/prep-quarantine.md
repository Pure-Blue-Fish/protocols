## פרמטרי מים (שבוע לפני!)

- [ ] ביופילטרציה: תסף אמוניה וודא שהמערכת מתמודדת *(אם לא מספק - יש שבוע לשיפור)*
- [ ] מליחות: השווה למקור הדגים, וודא פער מינימלי
- [ ] טמפרטורה: השווה למקור הדגים *(הפרש מקסימלי: 5°C)*
- [ ] pH: בדוק ורשום

## ניקיון והכנת מיכלים

- [ ] נקה מכלי דגים (כולל צנרת אוויר וחמצן)
- [ ] נקה וחטא את כל שטח הקרנטינה
- [ ] החלף רשתות יציאת מים בגודל מתאים לדגיגים

## ציוד לקבלת דגים

- [ ] מכלי אקלום דגים
- [ ] מד pH + טרמומטר
- [ ] מד מליחות
- [ ] סלייטים לרישום
- [ ] קנקנים לבדיקות מים
- [ ] צנרת אוורור וחמצון
- [ ] מד חמצן
- [ ] חומר סופח אמוניה
- [ ] רשתות יד בגודל מתאים
- [ ] דליי עבודה + משקל

## הכנת המעבדה

- [ ] ערכות מדידת אמוניה מוכנות *(בדיקה תיעשה בקרנטינה, לא במעבדה)*
- [ ] כלי ניתוח ומיקרוסקופ מוכנים *(בדיקות דגים רק בסיום הקליטה!)*
