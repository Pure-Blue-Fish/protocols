## שעות בדיקה ידנית

| משמרת | שעות |
| --- | --- |
| בוקר | 08:00, 12:00 |
| ערב | 16:00, 20:00 |
| לילה | 00:00, 04:00 |

## לפני הבדיקה

- [ ] קח מד חמצן ידני מעמדת הניהול
- [ ] בדוק שהמד מציג 100% באוויר *(סטייה מותרת: ±2%)*

## ביצוע הבדיקה

- [ ] מדוד בכל מיכל בנפרד
- [ ] רשום תוצאות *(אל תסתמך על מיכל אחד - כל מיכל שונה!)*

## רישום מאלקטרודות (כל שעתיים)

- [ ] קרא נתונים מהלוגרים או ממחשב הבקרה
- [ ] רשום בטבלה

## תחזוקה יומית

- [ ] נקה אלקטרודות
- [ ] נקה רשתות סביב הצינורות המגנים
- [ ] השווה קריאת אלקטרודה לקריאה ידנית

## בעיות ופתרונות

| מצב | פעולה |
| --- | --- |
| סטייה > 0.5 mg/L בין ידני לאלקטרודה | דווח לביולוג/מנהל מיד |
| סטייה > 10% | דווח לביולוג/מנהל מיד |
| מד ידני לא מראה 100% באוויר | נקה אלקטרודה, בדוק כיול |
| חמצן נמוך במיכל | בדוק מערכת חמצון, דווח מיד |
